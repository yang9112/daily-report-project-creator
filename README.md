# Daily Report Project Creator

> 🚀 基于AI摘要的技术博客自动化日报项目创建器

## 📖 简介

这个工具可以快速创建基于AI摘要的技术博客自动化日报项目。生成的项目包含：

- 智能采集多个技术博客的最新文章
- 使用人工智能生成文章核心观点摘要  
- 基于文章链接的智能去重机制
- 支持Markdown、HTML等多种输出格式
- 基于SQLite的轻量部署方案
- Docker容器化部署支持

## 🚀 快速开始

### 安装

```bash
npm install -g daily-report-project-creator
```

### 创建单个项目

```bash
daily-report-create my-tech-digest
```

### 批量创建项目

```bash
daily-report-batch
```

- ✅ 项目模板自动生成
